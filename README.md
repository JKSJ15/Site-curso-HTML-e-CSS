# 🌐 JK Desenvolvimento Web

Projeto desenvolvido durante um curso de **HTML5 e CSS3**, com foco no aprendizado de estruturação semântica, estilização com CSS e construção de um site institucional.

## Funcionalidades
- Página inicial com apresentação da empresa
- Menu de navegação entre páginas
- Área de clientes
- Seção de serviços
- Página sobre a empresa
- Página de contato
- Sessão de notícias
- Layout organizado em colunas
- Estrutura visual limpa e moderna

## Tecnologias utilizadas
- HTML5
- CSS3
- Normalize.css

## Estrutura do projeto
```bash
/
├── index.html
├── clientes.html
├── servicos.html
├── sobre.html
├── contato.html
├── css/
│   ├── normalize.css
│   └── estilo.css
└── img/
````
## Aprendizados:
Durante o desenvolvimento deste projeto foram aplicados conceitos como:
- Semântica HTML
- Separação entre estrutura e estilo
- Organização de arquivos front-end
- Estruturação semântica com HTML
- Organização visual com CSS
- Layout em colunas
- Boas práticas de front-end básico
- Construção de páginas institucionais
